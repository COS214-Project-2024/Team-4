\doxysubsection{Detailed Description}
A class to traverse through highways in a transportation system. 

The \doxylink{classTraverseHighway}{Traverse\+Highway} class provides methods to navigate through a list of highways, allowing movement to the next or previous highway and retrieving the current position.
