\doxysubsection{Detailed Description}
Manages resources, handles resource allocation and release, and notifies observers. 

The \doxylink{classResourceManager}{Resource\+Manager} class provides methods to allocate and release resources, set allocation strategies, manage budget, and notify observers about resource changes.
