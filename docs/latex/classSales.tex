\doxysubsection{Detailed Description}
Represents a type of tax that includes a base sales tax, an environmental levy, and a service fee. 

The \doxylink{classSales}{Sales} class inherits from the \doxylink{classTaxType}{Tax\+Type} class and provides methods to set and calculate the total sales tax based on a sale amount.
