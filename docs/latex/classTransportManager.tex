\doxysubsection{Detailed Description}
Manages various types of transportation objects. 

The \doxylink{classTransportManager}{Transport\+Manager} class uses a \doxylink{classTransportationFactory}{Transportation\+Factory} to create and manage different types of transportation objects such as highways, roads, buses, taxis, trains, and airports.
