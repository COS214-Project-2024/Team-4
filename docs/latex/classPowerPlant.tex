\doxysubsection{Detailed Description}
Represents a power plant that supplies power to buildings. 

The \doxylink{classPowerPlant}{Power\+Plant} class inherits from the \doxylink{classUtility}{Utility} class and provides methods to register buildings, supply resources, and generate electricity.
