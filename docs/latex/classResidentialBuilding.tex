\doxysubsection{Detailed Description}
Represents a residential building in the simulation. 

The \doxylink{classResidentialBuilding}{Residential\+Building} class inherits from the \doxylink{classBuilding}{Building} class and includes additional attributes and methods specific to residential buildings, such as residential units, comfort level, and methods to manage residents and calculate taxes.
