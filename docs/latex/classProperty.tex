\doxysubsection{Detailed Description}
Represents a property with specific attributes and behaviors related to property tax calculation. 

The \doxylink{classProperty}{Property} class inherits from the \doxylink{classTaxType}{Tax\+Type} class and provides methods to set tax rates, calculate property tax, and set additional fees.
