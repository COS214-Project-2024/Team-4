\doxysubsection{Detailed Description}
Represents a commercial airport. 

The \doxylink{classComercialAirport}{Comercial\+Airport} class inherits from the \doxylink{classAirport}{Airport} class and provides additional functionality specific to commercial airports, such as connecting roads and other commercial airports.
