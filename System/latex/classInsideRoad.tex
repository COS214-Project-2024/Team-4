\doxysubsection{Detailed Description}
Represents an inside road that can contain various transportation entities. 

The \doxylink{classInsideRoad}{Inside\+Road} class inherits from the \doxylink{classRoad}{Road} class and adds functionality to manage different types of transportation entities such as buses, taxis, airports, and trains.
