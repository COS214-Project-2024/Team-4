\doxysubsection{Detailed Description}
Interface for iterating over \doxylink{classTransportation}{Transportation} objects. 

The \doxylink{classTraverser}{Traverser} class provides pure virtual functions for incrementing, decrementing, and dereferencing iterators over \doxylink{classTransportation}{Transportation} objects. Classes that inherit from \doxylink{classTraverser}{Traverser} must implement these functions.
