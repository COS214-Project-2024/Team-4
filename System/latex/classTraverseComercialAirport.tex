\doxysubsection{Detailed Description}
A class to traverse through a commercial airport. 

The \doxylink{classTraverseComercialAirport}{Traverse\+Comercial\+Airport} class provides methods to navigate through different lists within a commercial airport. It maintains the current position and allows moving to the next or previous list.
